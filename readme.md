# Universal Bank Intelligence Hub

Client-friendly Streamlit dashboard for the Universal Bank personal loan case.

## Highlights
- Cleaner dark theme with tighter spacing
- Overview, Segments, Models, and Action Center tabs
- Personal loan targeting and cross-sell recommendations
- No Plotly, scikit-learn, or scipy dependencies

## Files
- `app.py`
- `requirements.txt`
- `.streamlit/config.toml`
- `UniversalBank.csv`

## Data snapshot
- Rows: 5000
- Columns: 14
- Loan acceptance rate: 9.6%
